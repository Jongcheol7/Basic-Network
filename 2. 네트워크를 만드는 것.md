# 2-1. 네트워크를 구성하는 기기
+ 라우터, 레이어2 스위치, 레이어3 스위치
+ 이 네트워크 기기들은 모두 데이터를 전송한다. 
+ 데이터 수신 : 전기신호 등 물리적 신호로 변환된 데이터들을 원래 디지털 신호(0,1)로 되돌리다.
+ 데이터를 보낼 곳을 결정 : 데이터에 부가된 제어정보를 참조하여 전송할 곳을 결정
+ 데이터 전송 : 데이터를 물리적인 신호로 변환하여 내보낸다.

# 2-2. 네트워크의 구체적인 구성
+ 인터페이스 : 네트워크 기기끼리나 PC, 서버 등을 연결하기 위해 각각의 기기에는 인터페이스가 갖추어져 있어야 한다. 현재 가장 일반적으로 이용되는 것은 이더넷 인터페이스이다. 이더넷 인터
  페이스는 포트라고 부를 수도 있다.
+ 전송매체 : 인터페이스를 서로 연결하는 케이블을 전송매체라고 한다. 물리적 신호가 전송매체를 통해 전달된다. 전송매체는 유선뿐만아니라 무선 전파인 경우도 있다.
+ 링크 : 전송매체를 통해 인터페이스끼리 연결된 상태를 링크라고 한다.
